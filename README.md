# 👥 Employee Management Application

A Python CLI application for managing employee data, attendance, and HR workflows with MySQL backend and Role-Based Access Control.

## 🔧 Tech Stack
- Python
- MySQL
- CSV Export (Tableau-ready)

## ✨ Features
- Role-Based Access Control (Admin, HR, Manager, Employee)
- Employee CRUD operations
- Attendance tracking with check-in/check-out times
- Salary management with net pay calculation
- Full audit logging of every action
- Tableau-ready CSV export for dashboards

## 🚀 How to Run
pip install mysql-connector-python
python employee_app.py
